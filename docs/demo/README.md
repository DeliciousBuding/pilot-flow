# 演示材料

## 演示脚本

### 场景：一句话创建项目空间

**演示步骤：**

1. 打开飞书群聊
2. 输入：`@PilotFlow 帮我准备答辩项目空间，成员唐丁，交付物是项目简报和任务清单，5月7日截止`
3. PilotFlow 展示执行计划
4. 用户回复「确认」
5. PilotFlow 自动创建：飞书文档、多维表格、任务、项目入口消息
6. PilotFlow 发送交付总结

**预期时间：** ~30 秒

### 场景：信息不完整

**演示步骤：**

1. 输入：`@PilotFlow 创建一个新项目`
2. PilotFlow 提示信息不完整，询问目标、成员、交付物
3. 补充信息后，PilotFlow 展示计划
4. 确认后执行
